# xrp-huskylense
This is a repository for using a Husky lens with the XRP robot. This is a non-official repository
___

To use this repository, place the files into the lib folder exactly as provided.
This implementation is configured to use QWIIC port 0 by default. If you need to use a different port, update the value in qwiic_huskylens.py on line 118.

For pin references, consult the labels on your XRP board or the SparkFun XRP Controller pinout documentation:
https://docs.sparkfun.com/SparkFun_XRP_Controller/hardware_overview/#motor-connectors
